# OpenMC-termux
Running openmc on Android phone

# Dependency List
- git
- cmake
- clang
- vim
- libhdf5
- libhdf5-static

# Installation
Download `Termux` from **Google Play Store**
```
pkg update
pkg install git
git clone https://github.com/AI-Pranto/OpenMC-termux.git
cd OpenMC-termux
chmod +x termux.sh
./termux.sh
```
