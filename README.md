# OpenMC-termux
Script for installing OpenMC on Android phone

# Dependency List
- git
- cmake
- clang
- vim
- libhdf5
- libhdf5-static

# Installation
Download `Termux` from **Google Play Store**
```
pkg update && pkg upgrade -y
pkg install git
git clone https://github.com/AI-Pranto/OpenMC-termux.git
cd OpenMC-termux
bash termux.sh
```
